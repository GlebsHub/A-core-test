# a-core

Клиент на **React** + **Vite** + **TanStack Router** с UI **@alphacore/ui-kit**: вход по GraphQL, дерево функциональных классов, карточка объекта (описание, характеристики, связи).

Запуск — **только локально** (зависимости и API через dev-прокси Vite).

## Клонирование и разворачивание в IDE

1. **Склонируйте репозиторий** (подставьте URL своего форка или оригинала):

   ```bash
   git clone <URL-репозитория>.git
   cd A-core-test
   ```

2. **Откройте папку проекта в IDE** (VS Code: *File → Open Folder…* и выберите `A-core-test`).

3. **Node.js** — установите LTS (например 20.x или 22.x), если ещё не стоит.

4. **Доступ к `@alphacore/ui-kit`** — скопируйте пример конфигурации npm и подставьте токен из тестового задания Alphacore:

   ```bash
   cp .npmrc.example .npmrc
   ```

   В `.npmrc` замените `${NPM_TOKEN}` на значение токена (или экспортируйте переменную окружения `NPM_TOKEN` до установки зависимостей — как указано в задании).

5. **Установите зависимости и запустите dev-сервер:**

   ```bash
   npm install
   npm run dev
   ```

6. В терминале появится адрес (обычно `http://localhost:5173`) — откройте его в браузере.

GraphQL в dev: прокси `/graphql` → `vite.config.ts`.
