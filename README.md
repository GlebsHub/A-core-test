# A-core

Клиент на **React** + **Vite** + **TanStack Router** с UI **@alphacore/ui-kit**: вход по GraphQL, дерево функциональных классов, карточка объекта (описание, характеристики, связи).

Запуск — **только локально** (зависимости и API через dev-прокси Vite).

## Клонирование и запуск

1. Склонируйте репозиторий и перейдите в папку:

   ```bash
   git clone <URL-репозитория>.git
   cd A-core-test
   ```

2. Откройте папку `A-core-test` в IDE.

3. В терминале из корня проекта:

   ```bash
   npm install && npm run dev
   ```

   Откройте в браузере адрес из вывода (обычно `http://localhost:5173`).
